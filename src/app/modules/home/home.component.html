Welcome to the Interdisciplinary Research Network Extension

IReN is a web application developed to facilitate the creation and access of case studies regarding the multi-faceted infrastructure damages of natural disasters in Puerto Rico. The web application allows for the creation, searching, and editing of case studies and was developed by students from the University of Puerto Rico to help the RISE-UP initiative document, collaborate, and share information that will help create more resilient future infrastructure in Puerto Rico.

Search Search documents in the database and apply different filters. Map Select filters for the documents and show the locations of it in a map. Timeline Select document to visualize the timeline of it. XvsY Plot Select filters and view a X vs Y plot to visualize the correlation.
